# 文章生成マルコフ連鎖

マルコフ連鎖で文章を生成するよ

Pythonで動作するよ

## 動作確認済み環境

- Windows 11(22H2) + Python 3.11, 3.12
- WSL Ubuntu 22.04.3(LTS) + Python 3.10.12

## インストール

1. このリポジトリをクローンする

```bash
git clone https://github.com/e6nlaq/word-ai.git
```

2. `requirements.txt`に書かれているライブラリをインストール

```bash
pip install -r requirements.txt
```

3. `main.py`を実行!
